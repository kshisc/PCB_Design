# Raspberry Pi Sensor ADC Interface PCB
This is a Raspberry Pi–compatible PCB shield designed in KiCad to enable analog sensor interfacing through an MCP3008 SPI-based ADC. The board integrates light and sound sensor input headers along with LED output circuitry. The design process included schematic capture, PCB layout, component selection, and routing of SPI and GPIO connections to ensure reliable communication between the Raspberry Pi and external sensors. After fabrication, the board was assembled using through-hole soldering and validated with a Python test program that reads sensor data via the MCP3008, outputs real-time light and sound values, and drives LED feedback based on configurable thresholds.
